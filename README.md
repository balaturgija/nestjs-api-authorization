## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## Install node_modueles

```bash
$ yarn
```
## Make shure your node version is set on v16.14.2
## Create .env file

```bash
  NODE_ENV=development
  PORT=
  HOST=
  DB_HOST=
  DB_PORT=
  DB_USER=
  DB_PASSWORD=
  DB_NAME=RoboBuilder
  JWT_SECRET=
```

## Running the app in development mode

```bash
# STEP1: create database container
$ docker-compose build

# STEP2: run database create
# Optional: because docker-compose will create databse
$ yarn run db:create

# STEP3: run migrations
$ yarn run db:migrate

# STEP4: run seeders
$ yarn run db:seed

# STEP5: start the app
$ yarn run start
```
